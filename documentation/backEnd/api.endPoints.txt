# API Endpoints Reference

## Auth — 7 endpoints
```
POST   /api/auth/signup
POST   /api/auth/login              (single endpoint, role-based redirect on frontend)
POST   /api/auth/verify-email
POST   /api/auth/forgot-password
POST   /api/auth/reset-password
POST   /api/auth/refresh-token
POST   /api/auth/logout
```

## Users — 6 endpoints
```
GET    /api/users/me                (own profile)
PUT    /api/users/me
GET    /api/users                   [admin] list/search customers
GET    /api/users/:id               [admin]
PUT    /api/users/:id/role          [admin] promote/demote
DELETE /api/users/:id               [admin]
```

## Products — 8 endpoints
```
GET    /api/products                (list, filters: category, price range, search)
GET    /api/products/:slug
POST   /api/products                [admin]
PUT    /api/products/:id            [admin]
DELETE /api/products/:id            [admin]
PATCH  /api/products/:id/stock      [admin]
GET    /api/products/:id/related
GET    /api/products/featured
```

## Categories — 5 endpoints
```
GET    /api/categories
GET    /api/categories/:slug
POST   /api/categories              [admin]
PUT    /api/categories/:id          [admin]
DELETE /api/categories/:id          [admin]
```

## Product Customization — 3 endpoints
```
POST   /api/customizations          (save a custom design: color/image/text)
GET    /api/customizations/:id
POST   /api/customizations/preview  (generate mockup preview image)
```

## Cart — 5 endpoints


## Wishlist — 3 endpoints
```

```

## Orders — 7 endpoints
```
POST   /api/orders                  (create order from cart, on checkout)
GET    /api/orders                  (own order history)
GET    /api/orders/:id
GET    /api/orders                  [admin] all orders, filterable by status
PUT    /api/orders/:id/status       [admin]
POST   /api/orders/:id/cancel
GET    /api/orders/:id/track
```
